# MinesweeperProject
Project Name: 
Tkinter Minesweeper

Filename: projectcurt.py

Installation & Requirements:
Python Version: Python 3.x.
Libraries: No external installations are required. The game uses standard libraries: tkinter, random, and sys.

Running the Project:
Open a terminal or command prompt.
Navigate to the directory containing projectcurt.py.
Run the script: projectcurt.py

How to Play
Reveal: Left-click on any square.
Flag: Right-click to place a red "F" on a square you suspect has a mine.
Chording: If you have flagged the correct number of mines around a revealed number, left-click that number to instantly reveal the remaining surrounding squares.
Winning: Successfully reveal all 119 safe cells (on a 12 x 12 grid with 25 mines).
